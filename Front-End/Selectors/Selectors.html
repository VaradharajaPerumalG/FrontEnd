Selectors

Selctors is classified into four five types and they are:

  1. Simple Selectors
    • Name
    • Id
    • Class

  2. Combinator Selectors
    • Descendant combinator (space)
    • Child combinator (>)
    • Next sibling combinator (+)
    • Subsequent-sibling combinator (~)

  3. pseudo-class Selectors

    4. A pseudo-class is used to define a special state of an element.

    Ocean Academy
    • Style an element when a user moves the mouse over it.
    • Style visited and unvisited links differently.
    • Style an element when it gets focus.
    • Style valid/invalid/required/optional form elements.

  4. Pseudo-elements Selectors

    5. A CSS pseudo-element is used to style specific parts of an element.

    • Style the viewbox behind a dialog box.
    • Style the markers of list items.
    • Insert content before or after an element.
    • Style the first letter or line, of an element

  5. Attribute Selectors

    6. The [attribute] selector is used to select elements with a specified attribute.
